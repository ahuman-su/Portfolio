# Portfolio BTS SIO

Portfolio statique de Titouan Pelou avec trois pages principales :

- `index.html` : présentation générale
- `bts.html` : compétences BTS SIO et document `E5.xlsx`
- `projects.html` : fiches projet détaillées avec liens GitHub quand ils existent

## Déploiement GitHub Pages

La version à publier est dans `docs/`.

Le site est public, mais un `robots.txt` et des balises `noindex` sont ajoutés pour
demander aux moteurs de recherche de ne pas référencer les pages.
